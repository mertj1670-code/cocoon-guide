# 🧩 Cocoon Strategy Guide

**https://cocoon-guide.notes.fit**

Complete multilingual strategy guide for **Cocoon** — the puzzle-adventure game by Geometric Interactive & Annapurna Interactive.

Walkthrough · Puzzle Solutions · Boss Strategies · Moon Ancestor Locations · Tiered Hints

---

## ✨ Features

- 🔍 **Real-time search** across all puzzles, bosses, and collectibles
- 🌐 **Multi-language** — English / 中文 / 日本語 with browser language detection
- 🕯️ **Tiered hints** — get a nudge before seeing the full solution
- 🗺️ **Interactive flow diagram** — clickable SVG roadmap of the entire game
- ✅ **Moon Ancestor checklist** — persistent progress with localStorage
- 🆘 **"Where am I stuck?"** — symptom-based quick-jump helper
- 📱 **Responsive** — works on desktop and mobile
- 🌙 **Dark theme** — inspired by the game's atmosphere

## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/your-username/cocoon-guide.git

# Open
open index.html
```

No build step. No dependencies. Single HTML file.
